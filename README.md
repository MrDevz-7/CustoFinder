# CustoFinder

Sistema de prospeccion inteligente de clientes para freelancers y agencias
de software: descubre negocios locales sin sitio web, los evalua con IA,
analiza su competencia local, y hace seguimiento del pipeline de ventas.

El proyecto esta dividido en dos carpetas hermanas:

- **[`backend/`](./backend/README.md)** - API en FastAPI + PostgreSQL.
  Setup, endpoints, y todo el detalle tecnico del backend.
- **`frontend/`** - Next.js + TypeScript (en construccion desde Dia 4).

Repo: https://github.com/MrDevz-7/CustoFinder - un commit por dia de
desarrollo.
